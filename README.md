## 💡 프로젝트 소개 & 회고

* **작성 배경**: 대학교 2학년 '웹 프로그래밍' 과목의 기말 과제로 제작되었습니다.
* **프로젝트 목표**: 한 학기 동안 배운 HTML, CSS, JavaScript의 다양한 기능과 요소들을 빠짐없이 실제 웹페이지에 적용해 보고 내 것으로 만드는 것을 목표로 했습니다.
* **회고**: 디자인 면에서는 다소 서툴고 아쉬운 부분도 있지만, 기초 웹 기술의 동작 원리를 몸소 익히고 직접 배포까지 시도해 본 뜻깊은 첫걸음이었습니다.

## 📌 주요 특징 및 구성 (Features)

* **여행지별 추억 갤러리**: 제주, 단양, 홍콩, 일본 등 주요 여행지별 콘텐츠 구성
* **기초 웹 기술 적용**: 수업 시간에 배운 태그, 스타일링, 스크립트 요소들을 종합적으로 활용
* **메뉴 네비게이션**: 각 페이지 및 미디어 자원으로 이동할 수 있는 기본 구조 구현
